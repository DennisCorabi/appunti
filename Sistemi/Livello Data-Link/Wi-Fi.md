# Wireless (802.11)


##### struttura
Serve un access point per collegarsi ad una rete wireless.

Gli access point sono  collegati fra loro con cavi in rame, per garantire una maggiore resistenza ai rumori ed una banda maggiore.

###### versioni

- 802.11 (legacy)
- 802.11a: 5 GHz
- 802.11b
- 802.11g
- 802.11n
